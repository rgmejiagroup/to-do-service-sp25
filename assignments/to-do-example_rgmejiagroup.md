## Example 2 – Create a Task using Postman

**Request**
- Method: POST  
- URL: http://localhost:3000/tasks  
- Headers:
  - Content-Type: application/json  
- Body (JSON):
```json
{
  "user_id": ...,
  "title": "...",
  "description": "...",
  "due_date": "...",
  "warning": "...",
  "id": ...
}

```

**Response**
```
{
  "user_id": 4,
  "title": "Fix API mismatch",
  "description": "Update example to include proper fields",
  "due_date": "2025-06-15T12:00",
  "warning": "-15",
  "id": 5
}
```
